Hints for table elements

nozebra

This table has an outputclass attribute value of nozebra.
This means that when the table is published, it will not have the alternating gray rows.
See more values available for the outputclass attribute here

headercolumn

This table has an outputclass attribute value of headercolumn.
This means that when the table is published, the first column of the table will be styled as a column.
See more values available for the outputclass attribute here

widetable

This table has an outputclass attribute value of widetable. This means that when the table is published, the table will push into the left margin in order to
be as wide on the page as possible without rotating either the table or the page.
See more values available for the outputclass attribute here

widetablenozebra

This table has an outputclass attribute value of widetablenozebra. This means that when the table is published, the table will push into the left margin in order
to be as wide on the page as possible without rotating either the table or the page and the alternating gray and white horizontal rows will not be present.
See more values available for the outputclass attribute here

rotatepage

This table has an outputclass attribute value of rotatepage. This means that when the table is published, the page will be rotated to landscape orientation.
If the widetable value is in the outputclass attribute as well, then the table will be taking up the maximum space available on the page.
See more values available for the outputclass attribute here

landscapetable

This table has an outputclass attribute value of landscapetable. This means that when the table is published, the table will be rotated on the page 90 degrees counter-clockwise. See more values available for the outputclass attribute here
