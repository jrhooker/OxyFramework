Hints for fig elements

basic figure

When figures don't fit easily in within the page, you have some options to make them work better.
You can expand the margin so that figure uses up more of the page, or you can rotate
the page into landscape mode, or you can do both.
Learn how to do this here [?]

Image Column

This image element has a value of column defined on its width attribute.
This is usually done in combination with adding the value wide to the
outputclass element of the containing fig element.
Learn more here [?]

wide

This figure has an outputclass attribute value of wide.
This means that when the figure is published, the figure will push
into the left margin in order to be as wide on the page as possible without
rotating either the table or the page. This is usually done in combination
with adding the value column to the width attribute of the
image element.
Learn more here [?]

rotatepage

This figure has an outputclass attribute value of rotatepage. This means
that when the table is published, the page will be rotated to landscape orientation.
It is also possible to use more of the page by pushing to the margins. Learn more here [?]
